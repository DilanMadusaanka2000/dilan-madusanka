# Modern Developer Portfolio

A sleek, responsive, and performance-optimized developer portfolio built with React.js, Tailwind CSS, and Framer Motion. This project features a vertical timeline, interactive project cards, technical skills visualization, a working contact form (UI), and full Dark Mode support.

## Features

-   **Responsive Design**: Mobile-first architecture using Tailwind CSS.
-   **Dark Mode**: System-aware and manual toggle with persistent state in `localStorage`.
-   **Animations**: Smooth entry animations and hover effects powered by `framer-motion`.
-   **Modern UI**: Glassmorphism effects, a primary branded color scheme (`#0f2cbd`), and Lucide icons.
-   **Modular Structure**: Highly reusable components focused on maintainability.

## Prerequisites

To run this project, you need:
-   `Node.js` (version 16.x or higher)
-   `npm` (version 7.x or higher) or `yarn`
